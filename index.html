How to add the key to your provider
RunPod
Vast.ai
1
Go to Settings
Log in to runpod.io, click your avatar in the top right and select Settings.
2
Add the SSH key
Scroll to SSH Public Keys, click + Add Key, give it a name (e.g. ComfyManager), paste the public key copied above and confirm.
3
Start the Pod
Create a new Pod — the key is included automatically. Copy the host and port from the Pod connection screen and enter them in the top bar of this app.
1
Go to Account

2
Add the SSH key

3
Start the instance
Once the instance is running, click Connect. You will find the host (e.g. ssh6.vast.ai) and port to enter in the top bar of this app.
